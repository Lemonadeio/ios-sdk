public class AlchemyDataNews

The AlchemyDataNews API utilizes natural language processing technologies to query the world’s news and blogs like a database.

  • getNews(start:end:query:failure:success:)

    Returns articles matching the query. If no query is given, simply returns a count of articles matching the start/end timeframe.

    Timeframe values as numbers are assumed to be in second, however some convenience values are recognized by the service. For a list of those values, visit http://docs.alchemyapi.com/docs/counts

    There are several specific query parameters and ‘return’ fields. A list can be found at: http://docs.alchemyapi.com/docs/full-list-of-supported-news-api-fields
